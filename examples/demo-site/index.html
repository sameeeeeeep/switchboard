This site runs on your Claude

No API key here. No inference bill for the operator. The prompt goes to the sidekick on your machine, runs on your model, and only the answer comes back — under per-site consent you control in the Relay extension.

The entire integration — the “5 lines”:

import { whenRelayReady } from "@relay/sdk";
const relay = await whenRelayReady();
await relay.connect({ reason: "demo chat" });
for await (const d of relay.stream({ prompt }))
  if (d.type === "text") out.textContent += d.text;
